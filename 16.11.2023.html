git Commands

1- git init ( To initialize a new Git repository, creating the necessary files and folders for version control )

2- git status ( provides information about the current status of files in the repo, showing which files are modified, staged, or untracked )

3- git log ( displays the commit history, shows a list of commits, their messages, authors, dates, and unique hash identifiers )

4- git add htmlfilename or any other file name ( add files to the current directory )

format : `git add `stages a specific file.

`git add .`stages all changes in the current directory.

5- git rm htmlfilename or any other file name ( remove files from the current directory )

format : `git rm `removes a specific file.

6- git commit -m "message" ( used to commit staged changes with with a descriptive message in it )

format : `git commit -m "Commit message here"`

7- git push ( pushing the repo after doing some changes in the repo)

8- git clone url ( to clone a repository )

format : `git clone ` clones the repository located at the specified URL

9- git bash
